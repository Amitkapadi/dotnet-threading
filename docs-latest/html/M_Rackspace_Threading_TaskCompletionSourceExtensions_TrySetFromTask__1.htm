Remarks
This method will return   if the Task TResult  provided by taskCompletionSource is already in one of the three final states: RanToCompletion, Faulted, or Canceled. This method also returns   if the underlying Task TResult  has already been disposed.

If task is in the RanToCompletion state, the specified result value is assigned to the TaskCompletionSource TResult  using the TrySetResult(TResult) method.

If task is in the Faulted state, the unwrapped exceptions are bound to the TaskCompletionSource TResult  using the TrySetException(IEnumerable Exception ) method.

If task is in the Canceled state, the TaskCompletionSource TResult  is transitioned to the Canceled state using the TrySetCanceled  method.
